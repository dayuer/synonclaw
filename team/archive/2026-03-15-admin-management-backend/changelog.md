# Changelog

## [1.1.0] - 2026-03-15

### 新增

- **Admin 管理后台** (`/admin`)
  - 🖥️ Dashboard 仪表盘 — 4 项核心指标（含计数器动画）、最近订单、系统状态
  - 📦 产品管理 — CRUD 操作 + 上架/下架切换
  - 🏢 客户管理 — 搜索/筛选（关键词、类型、状态）+ 详情（关联设备/订单）
  - 📋 订单管理 — 全流程状态跟踪 + 变更时间线
  - 👨‍💻 开发者网络 — 技能标签 + 认证等级管理 + 任务记录
  - ⚙️ 系统设置 — 系统信息 + OpenClaw Token/RPC 配置 + 主题设置

### 技术

- Admin 布局框架独立于官网（Sidebar + TopBar + Content Area）
- 600+ 行 admin.css 复用 tokens.css 设计系统
- Mock 数据层支持跨页面 CRUD 持久化
- 路由守卫：无效 admin 子路径自动重定向
- 响应式适配：768px/1024px 断点
