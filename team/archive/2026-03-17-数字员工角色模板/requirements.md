# 需求：数字员工角色模板（Worker Templates）

> PM 阶段 | 复杂度: **S**

## 背景

SynonClaw 的数字员工系统已具备完整的 CRUD + 设备绑定 + 成员分配能力。但目前创建数字员工时，需要手动填写角色名、描述、System Prompt 和插件配置，对普通管理员而言门槛较高。

用户反馈的股票分析场景（基本面分析师 / 情绪分析师 / 新闻分析师 / 技术分析师）是一个典型用例：每台 Claw 设备应被赋予一个明确的专业角色。

**核心设计原则**: 1 台 Claw = 1 个数字员工角色（1:1 绑定）。

## 用户故事

### US-1: 角色模板快速创建

> 作为租户管理员，我希望从预置的角色模板中一键创建数字员工，以便快速赋予设备专业能力。

**AC:**
- [ ] AC-1.1: 创建数字员工时，表单顶部新增「选择模板」下拉框
- [ ] AC-1.2: 选择模板后，自动填充 角色名 / 描述 / System Prompt / 插件 四个字段
- [ ] AC-1.3: 自动填充后用户仍可修改任意字段（模板只是初始值）
- [ ] AC-1.4: 提供「自定义」选项（默认），保持原有空白创建流程不变

### US-2: 内置行业模板集

> 作为平台运营者，我需要预置一批高质量角色模板，覆盖常见工作场景。

**AC:**
- [ ] AC-2.1: 预置至少 8 个角色模板，覆盖：技术开发 / 数据分析 / 财务 / 客服 / 股票分析 等场景
- [ ] AC-2.2: 每个模板包含: id / name / description / systemPrompt / plugins / icon
- [ ] AC-2.3: 模板按分组展示（通用 / 金融 / 技术），便于查找
- [ ] AC-2.4: Mock 数据中包含使用模板创建的数字员工示例
