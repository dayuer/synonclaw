# 需求：Admin 右下角角色切换器（管理员 ↔ 普通用户）

## 背景

当前 Admin 后台所有页面对所有用户可见。产品核心价值是"管理权与使用权分离"——管理员看到完整控制台（设备/员工/成员/运营），普通成员只看到工作台（数字员工卡片 + 对话）。

为方便演示和测试双视角，需在右下角添加一个浮动角色切换按钮，可在管理员和普通用户两个视角之间切换。

## 用户故事

作为**演示者/测试者**，我希望通过右下角的浮动按钮在管理员和普通用户视角之间切换，以便快速体验两种角色的不同界面。

## 验收标准

- [ ] **AC-1**: Given 任意 admin 页面, When 查看右下角, Then 都可见一个浮动角色切换按钮，显示当前角色
- [ ] **AC-2**: Given 当前为管理员视角, When 点击切换, Then 侧边栏仅显示「我的工作台」，路由自动重定向到 /admin/workspace
- [ ] **AC-3**: Given 当前为普通用户视角, When 查看侧边栏, Then 仅有「工作台」导航项，无管理功能入口
- [ ] **AC-4**: Given 当前为普通用户视角, When 点击切换回管理员, Then 恢复完整侧边栏，路由回到仪表盘
- [ ] **AC-5**: Given 浮动按钮, When 观察样式, Then 按钮有半透明背景和悬浮动画，不遮挡主内容
- [ ] **AC-6**: Given npm test, When 运行, Then 全部通过

## 优先级: P1
