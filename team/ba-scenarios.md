# GNB 操作面板 — 业务场景拆解

> BA 产出 · 2026-03-16

## 领域术语表

| 术语 | 定义 |
|------|------|
| GNB 节点 | 每台 SynonClaw 设备对应的网络层实体，包含 UUID、虚拟 IP、公钥等 |
| Passcode | 8 字符十六进制安全凭证 (`0xXXXXXXXX`)，用于节点间身份验证 |
| 私域子网 | 逻辑网络隔离单元，以 CIDR 定义地址空间，使用独立 Passcode |
| 节点注册 | 将新设备加入 GNB 网络的操作，需分配唯一 UUID 和虚拟 IP |

---

## 业务链路图

### 主链路 (Happy Path)

```
管理员 → 打开网络拓扑 → 点击「注册节点」→ 填写表单 → 提交
  → 节点出现在列表中 → 点击详情 → 编辑 Passcode → 保存
  → 切换到私域 Tab → 创建子网 → 节点自动归类到子网
```

### 异常链路

```
注册时 UUID 重复 → 表单校验失败 → 显示错误提示
注册时虚拟 IP 重复 → 表单校验失败 → 显示错误提示
Passcode 格式错误 → 提交被阻止 → inline 错误提示
删除子网时存在关联节点 → 删除被阻止 → 提示"仅空子网可删"
```

## 场景矩阵

### S1: 节点注册 — Happy Path (P0)
- **前置条件**: 管理员已登录，设备配额未满
- **动作**: 点击「注册节点」→ 填入 UUID `00002001`、IP `10.1.1.1`、类型 `normal`、加密 `arc4`、Passcode `0xAB12CD34`
- **预期结果**: 新节点出现在列表，状态 `online`
- **覆盖**: AC-1.1, AC-1.2, AC-1.3

### S2: 节点注册 — UUID 重复 (P0)
- **前置条件**: 存在节点 UUID `00001001`
- **动作**: 注册新节点 UUID 也为 `00001001`
- **预期结果**: 表单校验失败，提示"UUID 已存在"
- **覆盖**: AC-1.4

### S3: 节点注册 — 虚拟 IP 重复 (P0)
- **前置条件**: 存在节点 IP `10.1.0.1`
- **动作**: 注册新节点 IP 也为 `10.1.0.1`
- **预期结果**: 表单校验失败，提示"虚拟 IP 已被使用"
- **覆盖**: AC-1.4

### S4: Passcode 编辑 — Happy Path (P0)
- **前置条件**: 节点 `dev1` 已存在，Passcode 当前为 `0xA7B3C9D1`
- **动作**: 点击编辑 → 输入 `0xNEWPASS1` → 保存
- **预期结果**: 详情面板 Passcode 更新为脱敏后的新值
- **覆盖**: AC-3.1, AC-3.2, AC-3.3, AC-3.4

### S5: Passcode 编辑 — 格式不合法 (P0)
- **前置条件**: 编辑 Passcode 弹窗已打开
- **动作**: 输入 `hello` (非 hex 格式)
- **预期结果**: 校验失败，禁止提交
- **覆盖**: AC-3.2

### S6: 私域创建 — Happy Path (P1)
- **前置条件**: 管理员在私域 Tab
- **动作**: 点击「创建私域」→ 输入名称 `深圳办公室`、CIDR `10.1.0.0/24`、子网 Passcode `0x11223344`
- **预期结果**: 新私域卡片出现，CIDR 段显示正确，节点数为符合该 CIDR 的节点数
- **覆盖**: AC-4.1, AC-4.2, AC-4.3, AC-4.5

### S7: 私域删除 — 空子网 (P1)
- **前置条件**: 私域 `测试域` 存在，无成员节点
- **动作**: 点击删除
- **预期结果**: 私域被移除
- **覆盖**: AC-4.4

### S8: 私域删除 — 有成员节点 (P1)
- **前置条件**: 私域有 3 个成员节点
- **动作**: 尝试删除
- **预期结果**: 删除被阻止，提示"仅空子网可删除"
- **覆盖**: AC-4.4

### S9: 状态监控 — 节点详情概览 (P2)
- **前置条件**: 节点已存在且有关联隧道
- **动作**: 点击详情
- **预期结果**: 展示全部 GNB 属性 + 关联隧道数量 + 健康统计
- **覆盖**: AC-2.1, AC-2.2, AC-2.3

## 边界条件清单

| # | 条件 | 处理 |
|---|------|------|
| B1 | UUID 非 8 位 hex | 表单校验阻止提交 |
| B2 | 虚拟 IP 非合法 IPv4 | 表单校验阻止提交 |
| B3 | Passcode 非 `0x` + 8位 hex | 表单校验阻止提交 |
| B4 | CIDR 格式不合法 | 表单校验阻止提交 |
| B5 | 配额已满时注册 | 返回配额错误 |
| B6 | 删除非空子网 | 返回 false，提示不可删 |
| B7 | 空列表时的 UI | 显示空态占位符 |
