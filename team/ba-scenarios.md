# BA 场景拆解 — SynonClaw 官网重新定义

> 业务经理阶段 | 复杂度: L

## 领域术语表

| 术语 | 定义 |
|------|------|
| 安全私域网络 | 基于加密隧道的零信任虚拟组网，数据不经过公网 |
| 本地 AI 引擎 | 部署在本地硬件上的推理引擎，数据不出域 |
| 低代码平台 | 面向业务人员的可视化应用搭建平台 |
| Desk | 桌面级算力终端（个人/工作室） |
| Rack | 机柜级算力集群（企业/数据中心） |
| 数字员工 | 由 AI 驱动的自动化智能体 |

---

## 业务链路图

### 访客转化漏斗

```
首次访问 → Hero（5s 价值理解）→ 三支柱（能力认知）→ 社会证明（信任建立）
    ├── 个人/极客 → 产品页 Desk → 预定
    ├── 企业决策者 → 技术能力页 → 产品页 Rack → 预约演示
    └── 开发者 → 开发者页 → 注册 / 文档
```

### 信息架构

```
Navbar: [首页] [产品 ▾] [技术] [开发者] [关于]
                 ├ Desk
                 └ Rack

Footer: 产品 | 技术 | 开发者 | 公司
```

| # | 条件 | 预期行为 |

- 浏览器截图验证视觉效果
- 响应式断点测试（320px / 768px / 1440px）
